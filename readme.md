config file:
1. display.txt
  first line : font size
  seconfi line : duration
  
  example:
  14
  5
 
2. user_config.txt
  first line : appid
  second line : private key
  
  example:
  appid = xxxx
  p_key = xxxx
  
3. (optional) proxies.txt
  first line: your proxy name, both for http or https
  
  example:
  http://proxy.test.com:80
